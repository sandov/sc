1. actualizar repostitorios apt

2. Generar claves ssh locales

    y copiar el contenido de la clave publica

3. Logearme en Github desde la version firefox por defecto

    Agregar la clave SSH publica

4. Cerrar navegador
--------------------------------------------------------

5. descargar sc

sudo apt install git

mkdir Code
cd Code

git clone git@github.com:sandov/sc
cd sc/install_scripts

6. instalar firefox y neovim

./firefox_install.sh
./neovim_install.sh
./evil_firefox_install.sh

7. configurar Firefox

Configurar como navegador predeterminado
Agregar al panel
cambiar preferencias de firefox
instalar ublock origin
instalar "block site" de "ray", recommended check

8. Copiar configs a ~/.config

cd ~/Code/sc/
cp -r configs/* ~/.config/

-----------------------------------------------------------
8. Configurar teclado
	Agregar layout Spanish (Latin America)
	Untick country flag
	Swap ESC and Caps Lock 

9. Configurar touchpad

$ echo export MOZ_USE_XINPUT2=1 | sudo tee /etc/profile.d/use-xinput2.sh

$ mkdir -p ~/Documents/util/scripts/io_scripts/

$ cp io_scripts/current/14s_touchpad_scroll_speed.sh ~/Documents/util/scripts/

Add 14s_touchpad_scroll_speed.sh to autostart

10. Configurar parlantes

$ cp io_scripts/current/14s_speakers_toggle.sh ~/Documents/util/scripts/

follow the instructions in:
	https://gist.github.com/philipperoubert/a4dd97bd94c78a588147c044000a87da

11. Install Dropbox

https://www.dropbox.com/install-linux

sudo apt install ./dropbox_2020.03.04_amd64.deb

Don't use dpkg -i, because it doesn't install dependencies

Launch and login
